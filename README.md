# IBM Employee Attrition Prediction

## Project Overview

This project predicts employee attrition using machine learning techniques. The objective is to identify employees who are likely to leave the organization.

## Dataset

IBM Employee Attrition Dataset

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

## Machine Learning Model

Random Forest Classifier

## Model Accuracy

83.22%

## Features Used

* Age
* Department
* Distance From Home
* Education
* Education Field
* Environment Satisfaction
* Job Satisfaction
* Marital Status
* Monthly Income
* Number of Companies Worked
* Work Life Balance
* Years at Company

## Conclusion

The model achieved an accuracy of 83.22% and identified important factors contributing to employee attrition.# IBM_Employee-attrition-prediction
